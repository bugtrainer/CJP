# CJPHub Observatory — Data Access Tiers & API Limits

## Free Tier (Public Dashboard)
- Cost: $0/month
- Access: Live dashboard at https://www.cjphub.com
- Data: Real-time follower counts, platform intervention logs, narrative graphs, verified milestones, community stream
- Rate Limit: Browser-based access only, no API
- Support: Community (none)

## Researcher Tier
- Cost: $29/month
- Access: REST API with JSON responses
- Data: All Free tier data plus historical time-series, raw feed exports, sentiment distribution breakdowns
- Rate Limit: 10,000 requests/month, 1 API key
- Export Formats: JSON, CSV
- Support: Email support within 48 hours

## Institutional Tier
- Cost: $149/month
- Access: REST API with priority routing
- Data: Full historical archive, bulk narrative exports, bot probability scoring, credibility index datasets
- Rate Limit: 100,000 requests/month, 5 API keys
- Export Formats: JSON, CSV, Parquet
- Support: Priority email support within 24 hours, quarterly briefing reports

## Enterprise / Academic Tier
- Cost: Custom quote (Contact observatory@cjphub.com)
- Access: Dedicated API endpoints, webhook integrations, real-time streaming
- Data: Unlimited historical access, custom narrative tracking, white-label embedding
- Rate Limit: Unlimited requests, custom API keys
- Export Formats: All formats, custom schemas
- Support: Dedicated account manager, 99.9% uptime SLA, custom data pipelines
